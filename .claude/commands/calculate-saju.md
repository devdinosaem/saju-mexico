입력된 생년월일시로 사주 4기둥을 계산하고 기본 분석을 출력한다.

## 인자

$ARGUMENTS 형식: "YYYY-MM-DD HH:MM" (양력, 24시간제)
예: "1990-05-15 14:30"

## 절차

1. 만세력 엔진(packages/manseryeok)으로 4기둥 계산
2. 사주 엔진(packages/saju-engine)으로 기본 분석 수행
3. 결과 출력

## 출력 형식

```
═══ 사주 분석 결과 ═══

[사주 4기둥]
  년주: 경오(庚午) — 금(Metal) + 화(Fire)
  월주: 신사(辛巳) — 금(Metal) + 화(Fire)
  일주: ...
  시주: ...

[오행 분포]
  木(Madera): ██░░░ 2
  火(Fuego):  ████░ 4
  土(Tierra): █░░░░ 1
  金(Metal):  ██░░░ 2
  水(Agua):   █░░░░ 1

[일간]: ... (성격 핵심 키워드)
[용신]: ...
[십신 구성]: ...
```
