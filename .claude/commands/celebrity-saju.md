멕시코 유명인의 사주를 분석하여 Instagram 캐러셀 콘텐츠(5장)를 생성한다.

## 인자

$ARGUMENTS: 유명인 이름 + 생년월일 (선택적으로 시간)
예: "Canelo Álvarez 1990-07-18 14:00"
예: "Peso Pluma 1999-06-15"
예: "trending" (현재 멕시코 트렌딩 유명인 자동 선정)

## 절차

1. 유명인 정보 확인: 이름, 생년월일, 출생시(불명이면 12:00 추정), 출생지
2. 만세력 엔진으로 사주 4기둥 계산
3. 사주 엔진으로 오행/십신/용신 분석
4. agents/celebrity-saju.md 프롬프트 참고
5. 유명인의 공개된 성격/커리어/최근 이슈와 사주 분석 매칭
6. 5장 캐러셀 콘텐츠 생성 (각 슬라이드별 텍스트 + 이미지 디렉션)
7. 캡션 + 해시태그 생성
