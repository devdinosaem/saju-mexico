# 만세력(萬歲曆) 엔진

양력 생년월일시 → 사주 4기둥(년주/월주/일주/시주) 계산 엔진.
외부 API 없이 자체 계산. 이 패키지는 순수 계산만 담당하며, 해석은 saju-engine이 한다.

## 파일 구조

```
src/
  types.ts          — 천간/지지/오행/사주 타입 정의
  constants.ts      — 천간10, 지지12, 60갑자, 오행 매핑, 월건표
  solar-terms.ts    — 24절기 시각 데이터 및 조회
  lunar-calendar.ts — 음양력 변환 (필요시)
  pillars.ts        — 4기둥 계산 (핵심)
  fortune.ts        — 대운/세운 계산
  index.ts          — public API
data/
  solar-terms.json  — 절기 정밀 데이터 (1900-2100)
tests/
  pillars.test.ts   — 알려진 사주 100+개 검증
```

## 계산 로직 요약

### 년주 (Year Pillar)
- 입춘 이전 출생 → 전년도 간지
- 천간: (양력년 - 4) % 10
- 지지: (양력년 - 4) % 12

### 월주 (Month Pillar)
- 절기(節) 기준으로 월 결정 (절입 시각 이전 → 전월)
- 천간: 년간 기준 월건표(月建表)로 결정
- 지지: 인월(1월)~축월(12월) 고정 순환

### 일주 (Day Pillar)
- 기준일(1900-01-01 = 갑자) + 경과일수 → 60갑자 순환
- JD(율리우스일) 활용 권장

### 시주 (Hour Pillar)
- 출생시각 → 12지지 매핑 (23:00~01:00=자, 01:00~03:00=축, ...)
- 천간: 일간 기준 시두법(時頭法)으로 결정

## 절기 데이터

24절기 중 **절(節)** 12개가 월 구분 기준:
입춘(1월), 경칩(2월), 청명(3월), 입하(4월), 망종(5월), 소서(6월),
입추(7월), 백로(8월), 한로(9월), 입동(10월), 대설(11월), 소한(12월)

## 테스트 전략

알려진 사주와 대조 검증. 각 테스트 케이스:
```ts
{ birth: "1990-05-15T14:30+09:00", expected: { year: "경오", month: "신사", day: "...", hour: "..." } }
```
