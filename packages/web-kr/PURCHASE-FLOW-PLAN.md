# 사주TI 결제 → 리포트 플로우 기획

> 요약리포트 CTA "풀 리포트 받기" 클릭 후 흐름

## 플로우

```
[요약 결과] "✦ ₩9,900 · 풀 리포트 받기 ✦" 클릭
   ↓
[결제 확인] 결제 전 최종 확인 화면 (/purchase/[id])
   ↓ 결제 완료
[생성 중] 로딩 애니메이션 + 체크리스트 (/generating/[id])
   ↓ 리포트 생성 완료
[상세 리포트] 14개 섹션 풀 리포트 (/report/[id])
```
