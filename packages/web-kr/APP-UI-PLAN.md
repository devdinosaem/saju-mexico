# 사주TI 앱 UI 기획서 (확정)

> 모바일 웹 (max-width 390px). 기존 두들/크림+핑크/반말 디자인시스템 유지.
> 1탭 레이아웃: 카드 기반 (SETLOG 스타일). 2·3탭 내용: 1차 기획 채택 + 전체 고도화 포함.

---

## 앱 진입 구조

```
신규 유저                        재방문 유저
   ↓                                 ↓
랜딩 (기존 page.tsx)           탭 UI 직행 (로그인)
생년월일 입력
요약리포트 (무료)
"매일 운세 받기" 카카오 로그인 유도 ─→ 탭 UI 진입
```

탭 UI는 사주 입력 + 로그인 완료 상태의 홈. 기존 랜딩/result/report는 유지, 탭 UI는 그 위 레이어.

### A. 월운 달력 상세

**기본 기능:**
- 이번 달 달력 뷰
- 날짜마다 일진 도트: 상생 🟢 / 상극 🔴 / 중립 🟡
- 오늘 클릭 → 출석 도장 + 운세 오픈
- 지난 날짜 클릭 → 그날 열었던 운세 아카이브 재열람
- 미래 날짜: 🔒 잠금 (구독자는 미리 열람 가능)

**고도화:**
- **출석 스트릭**: 연속 출석 시각화 (3일 🔥 / 7일 ⭐ / 30일 🏆) + 스트릭 끊길 때 푸시 리마인드
- **스트릭 보상**: 7일 +3운기, 30일 +10운기 자동 지급
- **이번 달 에너지 예보**: 달력 헤더 위 1줄 ("6월은 금(金)의 달. 결단해!")
  > ⚠️ "주간 에너지 그래프"는 3탭에서 **5탭 마이로 이전**됨 → 마이 섹션 참조
- **오늘 카드 상세**: 에너지 방향 + 럭키컬러 + 럭키숫자 + 오늘의 조언

**오늘 운세 열람 후 상세 화면:**
```
  오늘 6월 23일 · 경진일주

  🟢 기운 좋은 날

  오늘의 에너지: 금(金) ↑↑
  럭키 컬러: 흰색 ⬜
  럭키 숫자: 4, 9

  "오늘의 한마디"
  "결단이 필요한 일, 오늘 결정해. 망설이면 기회 날아가."

  [내일도 보기 · 1운기]  [구독하면 한 달 무제한]
```

### A. 캐릭터 말풍선 시스템 — 고도화

**시간대별 멘트 (3구간):**
- 아침 (6~12시): 출발 에너지 ("오늘 뭔가 시작하기 딱이야!")
- 낮 (12~18시): 집중 에너지 ("지금 집중력 최고점. 중요한 것 지금 해")
- 저녁 (18~24시): 마무리 에너지 ("오늘 뭘 했든, 넌 잘했어")

**일주별 톤 차이 (필수):**
| 일주 | 톤 | 예시 |
|------|-----|------|
| 경진(庚辰) | 직설적, 강함 | "기회는 기다리는 게 아니야. 만드는 거지. 오늘도 GO!" |
| 을묘(乙卯) | 감성적, 따뜻함 | "오늘도 감성 충전 완료. 넌 그냥 존재만으로 충분해." |
| 임오(壬午) | 자유분방, 활기 | "오늘 어디 훌쩍 떠나고 싶지 않아? 작은 탈출이라도 해봐!" |
| 무진(戊辰) | 든든함, 중심 | "오늘도 묵직하게. 흔들리는 거 다 받아줄게." |

→ `ilju-types.ts`의 성격/대사 데이터 활용

**일진 연동 멘트:**
- 오늘 상생이면: "오늘 기운 찰떡이야!"
- 오늘 상극이면: "오늘 좀 조심해. 근데 넌 할 수 있어."

## 공통: 개인화 UX 시스템

요약 리포트 조회 후 앱 전반이 내 일주 + 오행에 맞춰 바뀜.

| 위치 | 개인화 내용 |
|------|------------|
| 1탭 헤더 | "경진일주, 안녕! 오늘 금(金) 기운 강해 💎" |
| 상세 리포트 카드 | 내 일주와 같은 유명인 먼저 노출 |
| 달력 색상 | 내 일주와 상생인 날 🟢 강조, 상극 🔴 주의 표시 |
| 밸런스 트래커 *(백로그)* | 내 용신 기반 목표 게이지 자동 설정 — FoodLogger.POLICY.md |
| 캐릭터 말풍선 | 일주 성격별 톤 + 오늘 일진 연동 멘트 |
| 상품 추천 순서 | 용신 오행 부족 시 관련 상품 우선 표시 |

## 카드 스타일 기준

```
배경색:  크림(#FEF9C3) 또는 흰(#FFFFFF)
테두리:  rounded-2xl (16px)
         두들 스타일의 #2D2D2D 1px 선 (선택적)
그림자:  없음 (테두리로만 구분 — 두들스러움 유지)
패딩:   카드 내부 16px, 카드 간격 12px
텍스트: 헤더 1줄 + 서브 1줄 + CTA 1개 = 최대 3줄
폰트:   본문 Pretendard/SUIT, 강조 Black Han Sans
```

## 추가 제안 (장기 로드맵)

1. **온보딩 플로우**: 요약리포트 후 "카카오 로그인하면 매일 운세 알림 드려요" CTA로 앱 탭 전환 유도
2. **Push 알림 전략**: "경진일주, 오늘 너한테 딱 좋은 날이야 🟢" 일주 맞춤 + 스트릭 끊길 것 같을 때 리마인드 (OneSignal 연동)
3. **궁합 카드 공유**: "오늘 나+지인 궁합 87%!" 카드를 인스타 스토리 공유 → 지인 유입
4. **일주 커뮤니티 티저**: 같은 일주끼리 익명 피드 ("경진일주들 오늘 어때?") — MVP 이후 구조만 열어둠
