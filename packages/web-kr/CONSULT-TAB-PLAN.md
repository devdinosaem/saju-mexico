# 상담 탭 기획서

> 위치: 2탭 (사주상품 우측)  
> 경로: `/v3/consult`  
> 과금: **0.1명태 / 1턴 (₩98)**  
> 엔진: **DeepSeek V3** (`deepseek-chat`)

---

## 핵심 컨셉

> **"내 사주를 가장 잘 아는 건 나야"**

일반 AI 사주 서비스는 외부 전문가가 내 사주를 읽어주는 구조다. 이 서비스는 반대다.

- 내 일주 캐릭터 = 내 에너지의 의인화
- 나의 사주를 완전히 내재화한 '또 다른 나'가 고민을 들어준다
- 조언하는 주체가 '나'이기 때문에, 무서울 만큼 정확하다는 체감을 준다

**포지셔닝 한 줄:**  
외부 점술가가 아닌, 내 에너지 자체와의 대화.

## 상담 무드 칩

대화 시작 전 어떤 맥락으로 상담할지 선택하는 컨텍스트 설정 UI.  
**메시지를 보내는 게 아니라, DeepSeek에 전달할 시스템 프롬프트 컨텍스트를 선택하는 것.**

### 칩 목록

| 라벨 | 두들 아이콘 | DeepSeek 컨텍스트 |
|------|-----------|-----------------|
| 연애 | DoodleHeart | 연애/이성 관계 관점으로 응답 |
| 직장 | DoodleSuitcase | 커리어/직장 관점으로 응답 |
| 재물 | DoodleCrystal | 재물/금전 관점으로 응답 |
| 직접 입력 | DoodleMagicWand | 사용자 입력 텍스트를 컨텍스트로 주입 |

### 칩 동작

```
연애/직장/재물 클릭
  → 해당 칩 활성화 (핑크 테두리 + 배경)
  → 다시 클릭 시 해제 (토글)
  → 채팅창에 "── {라벨} 상담 모드 ──" 구분선 삽입

직접 입력 클릭
  → 바텀 시트 팝업 오픈
  → 텍스트 입력 후 [적용 →]
  → 칩 라벨이 입력 텍스트(최대 8자+…)로 변경
  → 칩 활성화
  → 채팅창에 `── "{입력값}" 상담 모드 ──` 구분선 삽입
```

### 활성 칩 스타일

```
활성: bg-pink/10, border-pink, text-pink, font-bold
비활성: bg-white, border-charcoal/15, text-charcoal
```

### 모드 전환 알림 (시스템 메시지)

칩이 변경될 때마다 채팅 목록 마지막에 삽입되는 구분선 텍스트.  
말풍선이 아닌 가운데 정렬 + 양쪽 가로줄 스타일.

```tsx
{ role: "system", text: "연애 상담 모드" }
// 렌더: ─── 연애 상담 모드 ───
```

### 직접 입력 바텀 시트

```
┌──────────────────────────────────┐
│           ────                   │  ← 핸들바
│  어떤 걸 상담받고 싶어?             │
│                                  │
│  [텍스트 입력 (3줄)]               │
│  ex) 사업, 이사, 건강, 인간관계...  │
│                                  │
│  [취소]              [적용 →]     │
└──────────────────────────────────┘
```

- `fixed bottom-0`, 모바일 키보드 대응
- 바깥 터치 시 닫힘
- `inputMode="text"`, `autoFocus`

## AI 동작 명세

### 시스템 프롬프트 구성

```
[페르소나]
너는 {일주명} 일주의 에너지를 의인화한 캐릭터야.
이 사람의 사주를 완전히 알고 있는 '또 다른 나'로서 대화해.
외부 전문가가 아닌 내면의 목소리처럼 말해.
말투는 친근하고 직설적이며, 사주 개념을 자연스럽게 녹여서 써.

[상담 무드]
{선택된 무드 컨텍스트 — 없으면 생략}
ex) "지금 이 대화는 연애/이성 관계 관점에서 답변해줘."

[사용자 사주 데이터]
일주: {일주}
연주: {연주} / 월주: {월주} / 시주: {시주}
오행 비율: {목/화/토/금/수}
용신: {용신}

[응답 규칙]
- 답변은 3~5문장 이내로 짧게
- 사주 용어는 쉽게 풀어서
- 절망적인 말 금지, 항상 나를 믿는 방향으로
- 답변 끝에 되묻는 질문 1개 (대화 유도)
- 확정적 예언 금지 ("반드시 ~해야 해")
- 공포 조장 금지 ("올해 사고 조심해")
- 의료/법률 판단 금지
```

### 입력 제한

| 항목 | 값 |
|------|------|
| 유저 입력 최대 | **500자** |
| 입력창 UI | 1줄 시작, 최대 4줄 자동 확장, 글자수 카운터 표시 (유저 인지 필수) |
| AI 응답 최대 | **2000자** (시스템 프롬프트에 명시) |

### 대화 메모리

| 범위 | 처리 |
|------|------|
| 세션 내 window | **최근 8턴만 유지** (이전 턴 drop, 원가 캡 목적) |
| 세션 간 | 최근 3회 대화 요약 저장 → 재진입 시 연속성 제공 |

### 스크롤 정책

답변 생성 중 **자동 스크롤 없음** — 유저가 읽고 있는 위치를 유지한다.  
새 답변이 완료된 후에도 강제 스크롤하지 않는다.

### 과금 트리거

```
사용자 메시지 전송
  → 잔액 확인 (≥ 0.1명태)
  → 0.1명태 선차감
  → DeepSeek V3 API 호출
  → API 에러 시 자동 환불
```

## 메시지 타입

```ts
type Msg = { role: "user" | "char" | "system"; text: string }
```

| role | 렌더 방식 |
|------|---------|
| `char` | 좌측 말풍선 (흰 배경 + charcoal 테두리), 일주 캐릭터 아바타 |
| `user` | 우측 말풍선 (pink/15 배경) |
| `system` | 가운데 구분선 텍스트 `── {text} ──` (muted) |
