# 사주TI 상세 리포트 (/report/[id]) 기획

> 유료 결제 후 접근. 14개 섹션, 15,000자+ 풀 리포트.
> DeepSeek/Claude API가 생성한 텍스트를 친구 말투로 렌더링.

## 페이지 구조

### 헤더
- 이름 + 생년월일 + 일주 유형 카드 (요약 버전)
- 목차 (14개 섹션 가로 스크롤)

### 14개 섹션 (API 생성 텍스트)

| # | 제목 | 아이콘 | 내용 |
|---|------|--------|------|
| 1 | 너의 본질 | 🪞 | 일주 정체성, 성격, 강점/약점 |
| 2 | 오행 분포 | ⚖️ | 5가지 오행 균형, 불균형 영향 |
| 3 | 내면의 힘 | 💪 | 신강/신약 분석 |
| 4 | 십신 분석 | 🎭 | 가장 강한 십신과 영향 |
| 5 | 연애운 | 💕 | 파트너 유형, 연애 패턴, 시기 |
| 6 | 재물운 | 💰 | 재정 스타일, 풍요 시기 |
| 7 | 직업운 | 💼 | 적성 분야, 일하는 스타일 |
| 8 | 건강운 | 🏥 | 주의 부위, 건강 조언 |

### 하단
- 공유 CTA (카카오톡 + 인스타)
- "친구도 해보게 보내기"

## 톤 & 용어 기준

- 톤: 친구 반말, "카페에서 설명하는" 느낌 (학술적 X, 강의 X)
- 사주 용어: 한자 병기로 충분 (TermTooltip/ConceptCard 불필요 — 한국인은 기본 이해 있음)
- API가 이미 반말로 생성하므로 후처리 불필요
