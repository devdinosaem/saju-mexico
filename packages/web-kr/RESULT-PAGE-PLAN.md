## 용어 기준

- 사주 용어 별도 설명 불필요 — 한국인은 이미 기본 이해 있음, 한자 병기로 충분

## 데이터 소스

- **캐릭터 카드**: `ilju-types.ts` → `findIljuType(ilju, gender)`
- **오행 분포**: `manseryeok` → `calculateSaju()` → `fiveElements`
- **올해 세운**: `saju-engine` → `calculateYearlyFortunes()`
- **대운**: `saju-engine` → `calculateMajorFortunes()`
- **궁합**: `ilju-types.ts` → `bestMatch` / `worstMatch`

## 무료 vs 유료 경계

```
무료 (공유 가능):
  ✅ 캐릭터 카드 전체 (이름, 강점, 약점, 대사)
  ✅ 성격 요약 (description)
  ✅ 오행 밸런스 그래프 + 한줄 해석
  ✅ 찰떡 궁합 유형 (bestMatch)
  ✅ 올해 에너지 (어떤 해인지만)
  ✅ 대운 과거~현재

유료 (블러):
  🔒 올해와 나의 상호작용
  🔒 주의 궁합 유형 (worstMatch)
  🔒 대운 미래 전체
  🔒 14개 섹션 풀 리포트
```

## 공유 전략

```
1순위: 캐릭터 카드 캡처 → 인스타 스토리
2순위: 카카오톡 공유 → "너도 해봐" 링크
3순위: OG이미지 자동 생성 → URL 공유 시 미리보기

공유 시 보이는 것:
  - 유형 이름 + 이모지
  - 강점/약점
  - "120개 중 상위 X%"
  - "sajuti.com에서 확인"
```
